申请免费 AI 额度

本 App 的「AI 智能记账」走 OpenAI 兼容接口，默认接 ModelScope（魔搭社区） 的免费模型推理服务。绑定阿里云账号并实名后，每天约 2000 次免费调用，足够日常使用。

没绑定 / 没实名时调用会报：401 please bind your alibaba cloud account before use
0. 注册 / 登录 ModelScope

浏览器打开 modelscope.cn，用手机号或 GitHub 注册并登录。

1. 进入「绑定阿里云」入口

点右上角头像 → 菜单里点 Bind Alibabacloud（绑定阿里云）。这个页面的 Access Tokens（default / Write Permission）就是第 6 步要复制的 API Key。

2. 发起绑定

点 Link your Alibaba Cloud account to get free resources。

3. 登录 / 注册阿里云账号

已有账号直接登录；没有就点前往注册，免费注册一个。

4. 授权 ModelScope

阿里云弹出第三方应用授权页（申请方 ModelScope），点 授权。

5. 完成阿里云实名认证（开通额度的前提）

若显示 尚未实名认证（Not yet verified…），需先实名，免费 API 额度才会开通。

在账号认证页选 个人认证 → 个人支付宝认证（或扫脸认证），按提示完成。

6. 复制访问令牌 = API Key

回到 ModelScope 的 访问令牌（Access Tokens） 页，复制 default 那条 Write Permission 令牌（形如 ms-xxxx…）。令牌等同密码，别公开。

7. 在 App 里配置

在魔搭「模型库」选一个支持 API 推理的对话模型，复制完整模型 ID（带前缀），如 Qwen/Qwen3-30B-A3B-Instruct-2507。然后到「我的 → AI 智能记账 → 新增配置」填：

  • 来源 Base URL：https://api-inference.modelscope.cn/v1
  • API Key：第 6 步的 ms-… 令牌
  • 模型：完整模型 ID，如 Qwen/Qwen3-30B-A3B-Instruct-2507

保存后回记账页输入「今天打车35」点识别，能自动出账即成功。

常见报错
  • 401 …bind your alibaba cloud account…：没绑定/没实名 → 完成 1–5 步
  • 401 / Invalid token：key 填错或过期 → 重新复制令牌
  • 400 模型不存在：模型 ID 没带前缀 → 用完整 Qwen/…
  • 429 太频繁：稍等重试，或换一条配置/模型
